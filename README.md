# homebrew-tap

Homebrew tap for [@guillerDev](https://github.com/guillerDev)'s tools.

## Install

```sh
brew install guillerDev/tap/macthermal
```

or:

```sh
brew tap guillerDev/tap
brew install macthermal
```

## Formulae

- **macthermal** — macOS temperature & fan-speed analyzer that reads the SMC
  directly via IOKit. See [guillerDev/macthermal](https://github.com/guillerDev/macthermal).
