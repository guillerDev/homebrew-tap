# homebrew-tap

Homebrew tap for [@guillerDev](https://github.com/guillerDev)'s macOS tools.

## How to Add the Tap

To tap this repository to your Homebrew installation:

```sh
brew tap guillerDev/tap
```

---

## Formulae (Command-Line Tools)

### MacThermal (CLI)

A command-line macOS temperature and fan-speed analyzer that reads SMC keys directly via IOKit.

- **Source Code:** [guillerDev/macthermal](https://github.com/guillerDev/macthermal)
- **Installation:**
  ```sh
  # Direct install
  brew install guillerDev/tap/macthermal

  # Or after tapping:
  brew install macthermal
  ```

---

## Casks (GUI Applications)

These packages require `--cask` to install.

### MacPower

A live macOS power-consumption visualizer app (no root or helper tool required by default).

- **Source Code:** [guillerDev/macpower](https://github.com/guillerDev/macpower)
- **Installation:**
  ```sh
  # Direct install
  brew install --cask guillerDev/tap/macpower

  # Or after tapping:
  brew install --cask macpower
  ```

### MacThermal (App)

A menu-bar macOS temperature and fan-speed monitor application.

- **Source Code:** [guillerDev/macthermal](https://github.com/guillerDev/macthermal)
- **Installation:**
  ```sh
  # Direct install
  brew install --cask guillerDev/tap/macthermal

  # Or after tapping:
  brew install --cask macthermal
  ```
